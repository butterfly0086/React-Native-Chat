## Props

### <div class="label required basic">Required</div> client

The StreamChat client object

```jsx
import { StreamChat } from 'stream-chat'; 
import { Chat, OverlayProvider } from 'stream-chat-react-native';

const client = StreamChat.getInstance('api_key');

export const App = () =>
  <OverlayProvider topInset={0}>
    <Chat client={client}>
      {/** App components */}
    </Chat>
  </OverlayProvider>;
```

### closeConnectionOnBackground

When false, ws connection won't be disconnection upon backgrounding the app. To receive push notifications, its necessary that user doesn't have active websocket connection. So by default, we disconnect websocket connection when app goes to background, and reconnect when app comes to foreground.

| Type               | Default      |
| ------------------ | ------------ |
| boolean            | true         |

### i18nInstance

Instance of Streami18n class should be provided to Chat component to enable internationalization.

Stream provides following list of in-built translations:
1. English (en)
2. Dutch (nl)
3. French (fr)
4. Italian (it)
5. Hindi (hi)
6. Russian (ru)
7. Turkish (tr)

#### Example 1

Simplest way to start using chat components in one of the in-built languages would be following:

```jsx
import { Chat, Streami18n } from 'stream-chat-react-native';

const i18n = new Streami18n('nl');

<Chat client={chatClient} i18nInstance={i18n}>
 ...
</Chat>
```

#### Example 2

If you would like to override certain keys in in-built translation.
UI will be automatically updated in this case.

```jsx
import { Chat, Streami18n } from 'stream-chat-react-native';

const i18n = new Streami18n('nl');
i18n.registerTranslation('nl', {
 'Nothing yet...': 'Nog Niet ...',
 '{{ firstUser }} and {{ secondUser }} are typing...': '{{ firstUser }} en {{ secondUser }} zijn aan het typen...',
});

<Chat client={chatClient} i18nInstance={i18n}>
 ...
</Chat>
```

#### Example 3

You can use the same function to add whole new language.

```jsx
import { Chat, Streami18n } from 'stream-chat-react-native';

const i18n = new Streami18n('it');
i18n.registerTranslation('it', {
 'Nothing yet...': 'Non ancora ...',
 '{{ firstUser }} and {{ secondUser }} are typing...': '{{ firstUser }} a {{ secondUser }} stanno scrivendo...',
});

// Make sure to call setLanguage to reflect new language in UI.
i18n.setLanguage('it');

<Chat client={chatClient} i18nInstance={i18n}>
 ...
</Chat>
```

### style

You can pass the theme object to customize the styles of Chat components. You can check the default theme in [theme.ts](https://github.com/GetStream/stream-chat-react-native/blob/master/src/contexts/themeContext/utils/theme.ts)

Please check section about [themes in cookbook](https://github.com/GetStream/stream-chat-react-native/wiki/Cookbook-v3.0#theme) for details.

```jsx
import type { DeepPartial, Theme } from 'stream-chat-react-native';

const theme: DeepPartial<Theme> = {
  messageSimple: {
    file: {
      container: {
        backgroundColor: 'red',
      },
      icon: {
        height: 16,
        width: 16,
      },
    },
  },
};

<Chat style={theme}>
</Chat>
```
